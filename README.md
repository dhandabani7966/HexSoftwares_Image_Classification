# 🐱🐶 Cats vs Dogs - Image Classification

## 📌 Project Overview
A Deep Learning project that classifies images 
as Cat or Dog using CNN.

## 🛠️ Tools Used
- Python
- TensorFlow & Keras
- Matplotlib
- NumPy
- Gradio (Web App)

## 📊 Model Performance
- Training Accuracy: ~95%
- Test Accuracy: ~90%

## 🚀 How to Run
1. Open the .ipynb file in Google Colab
2. Run all cells
3. Upload any Cat or Dog image to predict!
